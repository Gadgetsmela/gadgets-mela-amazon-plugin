A secure Amazon wishlist importer with manual fallback and affiliate-ready product cards.

== Installation ==
1. Upload `gadgets-mela-amazon-importer` folder to `/wp-content/plugins/`.
2. Activate plugin in WordPress admin.
3. Go to `Gadgets Mela Amazon Importer > Settings`.
4. Add Amazon Access Key, Secret Key, Partner Tag, Marketplace, and Wishlist URL.
5. Use Wishlist Import or Manual Product Add pages.

== Shortcodes ==
- `[gm_amazon_products]`
- `[gm_amazon_product id="123"]`
